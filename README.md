# GitHub User Finder 
Автор: Панкова Полина

📄 Описание проекта
GitHub User Finder — это графическое приложение, написанное на языке Python. Оно позволяет искать пользователей на платформе GitHub по их имени и добавлять понравившихся в список избранного.

Проект создан для отработки практических навыков работы с API, создания графических интерфейсов и использования систем контроля версий.

⚙️ Технологии и инструменты

Язык программирования: Python 3.

Библиотеки:

Tkinter: для создания графического интерфейса (GUI).

Requests: для отправки HTTP-запросов к GitHub API.

Хранение данных: избранные пользователи сохраняются в локальный файл формата JSON.

Контроль версий: проект управляется с помощью Git.


🚀 Возможности приложения

Поиск пользователей: вводите логин пользователя GitHub и получаете его профиль.

Просмотр результатов: найденные пользователи отображаются в удобном списке.

Избранное: добавляйте пользователей в избранное одним кликом.

Сохранение данных: список избранных пользователей сохраняется в файл favorites.json и не теряется после закрытия программы.

Валидация ввода: приложение проверяет, чтобы поле поиска не было пустым.


🛠️ Как запустить проект (локально)

Установите Python (версии 3.8 или выше), если он у вас ещё не установлен.

Клонируйте репозиторий:

git clone https://github.com/<ваш_логин>/github-user-finder.git
cd github-user-finder

Создайте виртуальное окружение (рекомендуется):

python -m venv venv

Активируйте окружение:

На Windows:

.\venv\Scripts\activate

На Linux/macOS:

source venv/bin/activate

Установите зависимости:

pip install -r requirements.txt

Запустите приложение:

python main.py
